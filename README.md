# WRLD Plugin Store

A binary-only plugin catalog for **Minecraft PS4** (`CUSA00265`), compatible with version **1.02**.

## Publicly Available Plugins

- **Skin Browser**
- **Credits Mod Menu**

## Notes

Remote plugin icons are versioned beside their SPRX as `icon.png`. Icons must
be PNG RGBA/RGB, exactly `360x360`, and are declared in `catalog.xml` with
their raw GitHub URL, byte size, and SHA-256 digest.

The following components are **not** distributed through this public catalog:

- `wrld.sprx`
- Plugin Menu Manager
